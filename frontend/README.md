## Status
Placeholder — frontend development begins after backend is stable.

## Planned Stack
- React 18+ with TypeScript
- JMAP client for mail operations
- Tailwind CSS for styling
- Vite for build tooling

## Reference
While developing, use Roundcube at `http://localhost:8888` as a working webmail reference.
The JMAP endpoint is available at `http://localhost:8080/jmap`.
